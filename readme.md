# 🚀 Stock Monitor App
종목을 추가하고 선택하여 시세 및 상세 정보를 확인하며, 다양한 화면 모드를 통해 데이터를 효율적으로 모니터링하는 금융 도구입니다.

> **⚠️ 중요 안내 (필독)**
> * **투자 책임:** 본 프로그램은 참고용 모니터링 도구입니다. 모든 투자 결과에 대한 책임은 사용자 본인에게 있습니다.
> * **데이터 실시간성:** 실시간 체결 데이터가 아닌 약 15초 간격의 주기적 갱신 데이터를 사용합니다. 실제 시장 가격과 시간 차이가 발생할 수 있습니다.

## 📊 프로그램 개요
국내외 금융 데이터를 기반으로 종목 시세 및 상세 정보를 확인할 수 있는 데스크탑 모니터링 프로그램입니다.

## 🖥️ 화면 구성
* **좌측:** 종목 리스트
* **우측:** 선택 종목 상세 정보
* **하단:** 종목 입력창 및 기능 버튼

## 🛠 주요 기능

### 1. 종목 및 주요 지수 관리
* **추가:** 하단 입력창에 종목명/코드 또는 글로벌 지수 심볼을 입력 후 Enter 또는 `+` 버튼 클릭.
* **삭제 및 이동:** 종목 우측의 `×`(삭제), `▲`(위), `▼`(아래) 버튼 사용.
* **상세 정보:** 종목 클릭 시 상세 패널 표시.

**🇰🇷 국내(한국) 주식 및 지수 입력 가이드 (기본 네이버 금융 연동)**
한국 주식은 하단 입력창에 6자리 종목 코드 또는 종목명 검색을 통해 편리하게 추가할 수 있습니다.
* **종목명 검색 지원:** 입력창에 '삼성전자', '카카오' 등 기업명을 입력하여 검색 및 추가가 가능합니다.
* **종목 코드 입력:** 6자리 숫자(예: `005930`)를 입력하면 한국 주식 종목으로 자동 인식되어 처리됩니다.
* **국내 주요 지수 심볼:**
    * 코스피(KOSPI) 종합지수: `KOSPI` 입력
    * 코스닥(KOSDAQ) 종합지수: `KOSDAQ` 입력

**🇺🇸 해외 주요 지수 입력 가이드 (Yahoo Finance 연동)**
해외 지수를 모니터링하려면 하단 입력창에 심볼(`^` 접두사 포함)을 정확히 입력해 주세요. Yahoo Finance 공식 페이지에서 정확한 명칭을 찾을 수 있습니다.
* **S&P 500 지수:** `^GSPC` 입력
* **나스닥 100 지수:** `^NDX` 입력
* **다우존스 산업평균지수:** `^DJI` 입력
* **상해종합지수:** `000001.SS` 입력

### 2. 시세 정보 상세
* **리스트:** 현재가, 등락률 표시.
* **상세 패널:** 현재가, 등락률, 전일 종가, 시가, 고가/저가, 거래량, 시가총액, PER, PBR 등 제공.

### 3. 화면 모드 (UI 전환 및 중요 조작법)
* **Normal 모드:** 기본 화면 (리스트 + 상세 패널).
* **Excel 모드:** 리스트 중심의 간결한 화면 (상세 패널 숨김).
* **Stealth 모드:** 테두리 제거, 투명 UI, 위젯 형태 (바탕화면 드래그 이동 가능).

> **🔄 모드 전환 및 종료 규칙 (필독)**
> * Normal 모드에서 리스트 영역 우클릭 시 ➡️ Excel 또는 Stealth 모드로 전환됩니다.
> * Excel 또는 Stealth(Hide) 모드 상태에서 화면 우클릭 시 ➡️ 프로그램이 **즉시 종료(Close)**되므로 주의하시기 바랍니다.

### 4. 기타 편의 기능
* **검색:** KRX 데이터 기반 종목명 검색 기능.
* **설정:** UI 크기 변경(1x/2x), 다크 모드, 항상 위 설정 지원.
* **자동 저장:** 프로그램 종료 시 종목 리스트, UI 설정, 창 위치 자동 저장 (`stocks_list.json`, `stock_db.json`).

## ⚙️ 데이터 및 환경 안내
* **데이터 제공 방식**
    * 기본: 네이버 금융 데이터 (국내 주식 코드, 종목명 검색, 국내 지수).
    * 보조: Yahoo Finance (기본 데이터 실패 시 또는 해외 지수 심볼 입력 시 자동 전환).
* **권장 사용 환경**
    * 본 프로그램은 소수 종목 모니터링에 최적화되어 있습니다.
    * **권장 범위:** 10 ~ 15개 종목
    * **주의:** 15개 이상 등록 시 UI 반응 속도 저하가 발생할 수 있습니다.

## 🚨 자주 발생하는 문제 해결 (Troubleshooting)

* **특정 종목(예: KT, KT&G 등)이 영문 이름으로 검색되지 않을 때**
    * **원인:** 본 프로그램은 데이터의 완벽한 무결성을 위해 **한국거래소(KRX) 공식 법인 등록명**을 원본 그대로 사용합니다. 포털에서는 편의상 'KT'로 표기하지만, 실제 법적 공식 명칭은 '케이티'이기 때문에 발생하는 현상입니다.
    * **해결 방법:** 공식 한글 명칭(예: `케이티`, `케이티앤지`)으로 검색하시거나, 가장 확실하고 빠른 방법인 **우측 '코드검색' 칸에 6자리 종목코드(예: `030200`)를 직접 입력**하여 추가해 주시기 바랍니다.

* **종목 가격 위치에 "ERR" 문구가 지속적으로 표시될 때**
    * **원인:** 일시적인 네트워크 지연이거나, 내부 설정 파일 데이터가 엉켰을 가능성이 높습니다.
    * **해결 방법:** 프로그램이 실행되는 폴더에 생성된 **`stocks_list.json` 파일을 삭제한 후 프로그램을 다시 실행**해 보세요.
    * **💡 주의사항:** 해당 파일을 삭제하면 기존에 등록해 둔 종목 리스트와 화면 설정(다크모드, 창 위치 등)이 초기화되므로 유의하시기 바랍니다.

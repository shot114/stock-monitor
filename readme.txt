중요 안내 (필독)

본 프로그램은 주식 및 금융 데이터를 조회하는 참고용 모니터링 도구입니다.
본 정보는 투자 판단의 참고 자료이며, 모든 투자 결과에 대한 책임은 사용자 본인에게 있습니다.

본 프로그램은 실시간 체결 데이터가 아닌 주기적 갱신 데이터(약 15초 간격)를 사용합니다.
표시되는 가격은 실제 시장 가격과 시간 차이가 발생할 수 있습니다.

데이터 갱신 주기: 약 15초 (실시간 데이터 아님)

1. 프로그램 개요

Monitor는 국내외 금융 데이터를 기반으로 종목 시세 및 상세 정보를 확인할 수 있는 데스크탑 모니터링 프로그램입니다.

2. 화면 구성
좌측: 종목 리스트
우측: 선택 종목 상세 정보
하단: 종목 입력창 및 기능 버튼

동작 결과
입력한 종목이 리스트에 추가됨

5. 데이터 제공 방식
5.1 기본 데이터
네이버 금융 기반 데이터 사용
5.2 보조 데이터
Yahoo Finance 데이터 사용
기본 데이터 실패 시 자동 전환
5.3 코드 처리 방식
6자리 숫자 입력 시 한국 종목 코드로 자동 처리
일부 종목은 Yahoo Finance 형식으로 자동 변환
6. 데이터 갱신 방식

본 프로그램은 실시간 데이터가 아닌 주기적 갱신 방식을 사용합니다.

갱신 주기: 약 15초
선택 종목은 추가로 상세 정보 갱신 수행
네트워크 및 API 상태에 따라 지연 발생 가능
7. 화면 모드
7.1 Normal 모드
기본 화면

7.2 Excel 모드
리스트 중심 화면
상세 패널 숨김
7.3 Stealth 모드
창 테두리 제거

드래그 이동 가능
7.4 모드 복귀
리스트 영역 우클릭 시 Normal 모드로 복귀

하단 입력창에서 종목명 검색 가능
KRX 데이터 기반 검색 결과 표시
검색 결과 클릭 시 종목 추가
9. 설정 기능

항상 위 설정

11. 입력 관련 안내
11.1 종목 추가 입력
Enter 키 입력으로 종목 추가 가능
11.2 입력 동작 특성
입력값은 별도 검증 없이 즉시 반영됨
존재하지 않는 값도 리스트에 추가될 수 있음
검색 결과 없음 항목도 표시될 수 있음
12. 권장 사용 환경

본 프로그램은 소수 종목 중심의 모니터링 용도로 설계되었습니다.

권장 사용 범위: 10 ~ 15개 종목
15개 이상부터 UI 갱신 및 렌더링 부하 증가 가능
20개 이상 등록 시 환경에 따라 반응 속도 저하 발생 가능
한 줄 요약

종목을 추가하고 선택하여 시세 및 상세 정보를 확인하며, 다양한 화면 모드를 통해 데이터를 효율적으로 모니터링하는 금융 도구입니다.
